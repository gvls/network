## 2 `response head`
`Content-Base` :

`Content-Encoding` : 
`server` use which method to compress object of `response` 

`Content-Language` :
use which language to understand `response` 

`Content-Type` : 
* pre define
`text/plain` 
`text/html` (default)
`text/css` 
`image/jpeg` 
`image/png` 
`image/svg+xml` 
`audio/mp4` 
`video/mp4` 
`application/javascript` 
`application/pdf` 
`application/zip` 
`application/atom+xml` 

* custom (`MIME type`)


`Content-length` :
length of `response` 

`Etag` :
mark of `url` to `标志` resource of require if change

`Age` :
time which `实体` save in `response` 

`server` :
information of software which `server` use

`Allow` :
`server` support which `request method`
